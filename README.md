### 📊 Engenharia de Custos: Modelagem de Perdas e Custeio de SKUs
Este dashboard foi desenvolvido para traduzir dados de engenharia em impacto financeiro real, permitindo a priorização de ações de melhoria contínua.

![Dashboard de Custos](./Engenharia de Custos e Modelagem de Perdas.png) 

**Destaques técnicos:**
* **Ferramenta:** Microsoft Excel (VBA / Power Query)
* **Objetivo:** Identificar o custo de oportunidade por família de SKUs críticos.
* **Impacto:** Visibilidade imediata de perdas ocultas no processo produtivo.
